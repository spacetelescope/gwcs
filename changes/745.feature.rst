Expanded `gwcs.spectroscopy.WavelengthFromGratingEquation` to support FITS ``-GRA``/``-GRI`` grating coordinates by applying the grism correction terms to the input incident and refracted angle sines while preserving its original two-input grating-equation behavior.
Added `gwcs.spectroscopy.RefractedAngleSineModel`, a full `~astropy.modeling.Model` subclass that computes the pixel-to-refracted-angle-sine mapping required for FITS ``-GRA``/``-GRI`` spectra. The model accepts ``pixel`` as its single input and returns the sine of the refracted angle for use as the second input to `~gwcs.spectroscopy.WavelengthFromGratingEquation`.
